Wood Styles
--------------------------------------------------------------------------------------------------------
- INSTRUCTIONS: 

To add the style pack, make double click in the "Wood Styles" .asl file and go to
the "styles palette" in Photoshop.

- All of "Wood Styles" shown in the preview are actually a combination of two different layers.To get the correct effect, you must find the correct match. (for example - first use "Wood 1_base", duplicate layer and use "Wood 1")

Please See Psd File .

- RECOMMENDATION: 
If you want the same effect shown in the Wood Styles PSD file and dont have interest about install
the complete pack into your Photoshop, just drag and drop the chosen text to your own design and edit them!
